Steps to run:

install pip:
 - https://pip.pypa.io/en/stable/installation/ 
or https://www.geeksforgeeks.org/how-to-install-pip-on-windows/ 

install required packages via pip:

pip install PyQt5       -used qt for the gui design
pip install psycopg2    -psycopg2 for the db connection

setup credentials in CRUD-GUI.py
fill these in the code: 

DATABASE_NAME = ""
DATABASE_USER = ""
DATABASE_PASSWORD = ""
DATABASE_HOST = ""
DATABASE_PORT = ""

run application:

python CRUD-GUI.py
